数当てゲーム

このゲームは、1から10までのランダムな整数をプレイヤーが予想して当てる数当てゲーム。 プレイヤーは入力ボックスに自分の予想した数を入力し、「回答」ボタンをクリックすることで予想を送信する。 予想のたびに回数がカウントされ、最大3回まで挑戦できる。 予想が当たれば「正解です、おめでとう!」と表示されゲームは即終了する。 また、外れた場合は「もっと大きい」または「もっと小さい」とヒントが表示され、プレイヤーが次の予想をしやすいように工夫されている。 3回目の予想が外れた時点でゲーム終了となり、正解の数が表示される。 inputやbutton要素を正しく操作して、回数や結果を表示する処理に苦労した。 回数や結果の表示をリアルタイムで更新するために用いた DOM の操作にも苦戦した。

数当てゲームのページ: kazuate.html

各授業回の学習事項

第1回 WWW と HTML

webページを構成するHTML文書の使い方を学んだ。 また、pやstorongのブロック要素とインライン要素について学んだ。 HTML全体の構造を組み立てるうえで、これらの要素がどのように使われるか、それぞれが持つ役割についても学ぶことができた。

演習の成果: elements.html

第2回 CSS による Web ページの装飾

CSSとHTMLを組み合わせてWebページのデザインを整える方法について、3通りの記述方法を学んだ。 まず1つは外部スタイルシートで、CSSを別ファイルで用意し、HTMLファイルの head 内で link タグを使って読み込む方法。 2つ目は内部スタイルシートで、HTMLファイルの head 要素内に style タグを作成し、その中に直接CSSを記述する方法。 3つ目はインラインスタイルで、HTMLタグのstyle属性に直接CSSを書く方法。

演習の成果: browser-intro.html

第3回 JavaScript の基本

Webページに動きのある機能を実現するために、JavaScriptを使用する方法を学んだ。 JavaScriptは、HTMLファイルとは別にjsファイルを作成し、それをHTMLの head 要素や body 要素の末尾に script タグを使って読み込む。 こうすることで、HTMLと動作JavaScriptを分離し、Webページを作成できる。 。

演習の成果obj-practice.html

第4回 DOM と JavaScript

ボタンを押すことでページの内容が変化する処理をJavaScriptで学んだ。 JavaScriptとDOMを組み合わせて使うことが重要である。 DOMは、HTML文書内の各要素をJavaScriptのオブジェクトとして対応づける仕組みであり、これを利用することでページの要素を直接操作できる。

演習の成果: browser-intro.html

第5回 イベント処理

データをコンソールに出力するのではなく、ページ上に表示できるようにした。 これにより、ユーザーが画面上で検索結果などの情報を確認できるわかりやすいwebページが完成した。。 ページ上に要素を表示するために、要素名を使って新しい要素を作成する。 そして、要素を用いて必要な属性を設定し、内容を追加する。 また、不要になった要素は 要素.remove() で削除することができる。

演習の成果: elements.html
